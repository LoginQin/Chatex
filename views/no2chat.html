No.Chat - WebIM

   NoChat（我暂时还没想好名字;)）, 目标是构建高性能，可扩展的WebIM系统，使用Nodejs + Socket.IO + Redis构建，当前是Github上发布的版本。

关于聊天扩展

   NoChat可以利用第三方API很方便的扩展功能，正如您看到和未看到的：

代码高亮（Google Pretty Code） 实时搜索加入音乐（虾米音乐） UC豆瓣音乐台 中英双语实时翻译(有道翻译) 您还可以扩展更多...

聊天代码高亮

   聊天内容启用代码高亮，通过 # + 代码类型 + 空白 开启，像#js, #html, #c++, #java, 您也可以笼统的使用#code，例如在聊天中这样输入下面这段代码：

#js
var func = function(a) = {
    return  a + 5;
};

   系统会调用扩展的Google代码高亮模块，高亮代码显示给双方，这对开发者之间交流很有用哦

#js
var func = function(a){
    return a + 5;
}

实时插入虾米音乐

   您是否想介绍的一首喜欢的歌给对方？通过虾米音乐API，可以很方便分享喜欢的音乐给对方;)， 通过下面的格式开启实时搜索并嵌入音乐

 #音乐   #music   #xiami

   除了空白，你也可以用“=”或“：”， 如#music=海阔天空

   当然“海阔天空”有Beyond版和信乐团版，那么你应该这样输入：启用命令 歌名 歌手 ， 以保证搜索正确

#音乐 海阔天空 beyond

 这样就可以实时加入一个对应的虾米音乐盒给对方收听啦！

UC豆瓣电台

  想随机听听音乐？使用输入下面的豆瓣命令试试:)，不过听的歌就不是一样的啦，很多功能你都可以通过给自己发信息尝试。

#douban

有道中英互译

  聊天对象是老外,自己不太懂英文？通过#youdao, #cn, #en, #翻译， 加空白（tab，空格，回车等）后输入您的内容，把有道翻译请出来帮你翻译！

#翻译
你好啊，我很喜欢你的创意。
有道API支持中英互译，即当你启用这些命令后，有道会检测你输入的类型，例如输入英文，会帮你翻译成中文，输入中文，会帮你翻译成英文。跟老外聊天一样是去查字典，不如直接使用吧^_^

不熟悉怎么办？

  您可以通过给自己发消息，试验这些功能，当你熟悉后就像使用VIM，打字的时候可以不用动鼠标加入这么多好玩的，点击右边的微型窗口打开聊天工具，尝试一下吧，祝你愉快！

使用IE6?

       我明白如果作为一个推广项目，国内不支持IE6，后果非常惨烈，但是作为开发者，我们都有推动互联网前进的责任和使命，当你支持IE6，意味着IE6的用户们可以高枕无忧的使用你的应用，他们不会去想着升级浏览器，而开发者将会每次都要对付这个落后的浏览器，这种恶性循环都是由于目光短浅且唯利是图的国内商家们造成的，根本没有将推动互联网前进作为使命，如果国内巨头都有乔布斯推动互联网前进的霸气(敢于最先废除软驱、光驱、Flash、现在苹果的Mac Pro又废除以太网接口了)，国内互联网也不至于这么落后，特别是网银，资深互联网用户不放弃IE大都因为网银只支持IE系列，如今互联网已经走进千家万户，当IE6用户每次打开一个网页都出现错乱和提示升级浏览器，由于他已经有了上网的需求，他将不得不升级浏览器。写IE6 的CSS Hack并不难，由于此项目为个人实验项目，为了互联网更好的发展，以及帮助前端开发者们摆脱困境，我决定加入反IE6的队伍中，IE6用户请升级您的浏览器，您将会获得更好的体验，该WebIM在HTML5浏览器会自动选用WebSocket新一代的高性能通讯方式。一个人的力量是渺小的，当开发者们联合抵制IE6，IE6用户总有升级浏览器的一天。就像腾讯发展到今天已经不会再支持旧的QQ，强制用户升级到新的QQ，为广大QQ用户提供统一体验，大家有目共睹。

浏览器支持：IE7+ , Chrome8.0+ , FireFox 4.0+, 部分特效仅在CSS3支持的浏览器实现
