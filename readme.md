## 关于NoChat
   NoChat研究高性能Web实时应用架构而实现的高性能，可扩展的WebIM系统。采用Nodejs + Socket.IO + Redis 实现。

### 小历史
   2012年开始时名字还没想好 : )，当时就叫NoChat(Node Chat 或者 Not Only Chat), 目前申请了个域名和空间[Chatex.im](http://chatex.im)

   当时这个版本测试运行在Nodejs的PaaS上(Nodejitsu)，[http://chatex.jit.su/chat](http://chatex.jit.su/chat)

   后来该网站收费后把我运行的项目停了,这个地址已经无法运行,
  
   工作后不断有改动, 已经暂停更新到github.com, 目前运行在[http://chatex.im](http://chatex.im)做演示
